# XGui
Run `./xgui` with json input

Example: `./xhead {"foo": 100, "bar": 200}`

# Project Structure
```
├── Other
|   ├───META-INF
│   ├── custer.jar -- compiled program to be run through the shell script
│   └── src
│       ├── main -- all main source files
│           └── XGui.scala
├── README.md
└── xhead.sh
```
